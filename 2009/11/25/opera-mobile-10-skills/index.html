Opera Mobile 10 设置技巧 & 疑难问题

Nov 25th, 2009 | Comments

S60浏览器王者！Opera Mobile 10 终于出现！！

S60上的重量级浏览器Opera Mobile终于在近日放出了最新的10.0 beta 版，其界面与最近才出现的Opera Mini 5 一模一样，大图标、平滑滚动效果，滑动菜单等现代元素使得其界面非常有Iphone的感觉、桌面级的页面渲染效果更能带来如同电脑一般的浏览体验（感觉我有写枪文的潜力）。个人推荐下载OPDA汉化组的Opera Mobile 10 汉化版，据说有的版本不能很好显示东亚文字，但是这个版本没有任何问题：

Opera Mobile 10 貌似是为大屏幕的用户而设计的，我们320×240分辨率的屏幕用起来感觉是有些别扭啊。

以下是一些设置上的技巧：

  • 无法在文本框里面输入中文…… 其实进入设置里面，高级-把那个网址自动补全的功能关了就行了。

这样就能调用输入法输入文字了。

  • 怎么Opera Mobile 10浏览网页时字体这么小？不能改？ 解决方法是在地址栏输入about:config，出现一个Otaku们都非常熟悉的页面了：在Quick Find搜索框里面搜索font size 就会出现搜索结果，出现两个属性值：Maximim Font Size和Minimum Font Size，把Minimum的这个改成15或者16吧，总之随意改到您尽兴为止。
  • 浏览Wap站点的时候，屏幕两边的空白好大？ [caption id=”” align=”alignnone” width=”240” caption=”空白太大”][/caption] 仿佛Opera在告诉我们，赶快换大屏幕的手机吧，否则就囧死你的眼睛。不过既然我宅过一段时间，我才不吃这一套。仔细研究了下发现这些Wap页面使用了 一种类似于Html标准的语言，名叫Wireless Markup Language，缩写为WML，不过这可是比HTML要标准得多的XML语言，就像我们下在电脑上使用浏览器载HTML文档浏览普通页面一 样，Opera浏览器使用Wap协议下载这些WML文件并显示出来：其源代码类似于这种：

[xml]<?xml version="1.0"?> <!DOCTYPE wml PUBLIC "-//PHONE.COM//DTD WML 1.1//EN" "http://www.phone.com/dtd/wml11.dtd" > <wml> <card id="main" title="First Card"> <p mode="wrap">This is a sample WML page.</p> </card> </wml>[/xml]

至于显示的效果自然是用一个CSS来控制了，这个CSS文件的奇妙作用就是让没有生气而且乱作一团的一坨文字变得清晰易读，它能很好的控制字体的大小，标题的颜色，页面的背景色，段与段之间的空白……太多了，Opera想让我们换大屏幕手机，所以在CSS中把文字两边的空白留得太多了。没关系。只要稍微改动一小下下就可以了，打开 <你的Opera所在的盘符>systemappsOperaMobilestyleswml.css，用你手机上的文本编辑器或者弄到电脑上都可以（可能需要把Windows资源管理器设置为“显示所有文件和文件夹”），看到以下一坨，加粗的部分就是可能需要改的地方：

[css]/————————————/ @charset "utf-8"; / Stylesheet for WML documents / / Copyright 2009 Opera Software / @namespace url("http://www.wapforum.org/2001/wml"); wml, template, card, p, br, pre { font: 15px sans-serif; /字体大小，推荐改成15px或者16px / display: block; } wml { color: black; background: white; padding: 0; /白色边框的大小，建议设为0 / } template { background: #DDDDDD; padding: 5px; margin-bottom: 0.3em; }

/————— 中间省略一部分… —————/

card { text-align: left; background: #EEEEEE; /正文所在区域的背景颜色，要改成白色的话就设为#FFFFFF / padding: 1em； /推荐改成0.5em / display: block; } /—————————————-/[/css]

具体的数值可能不同，但是这些名字应该是一样的。

1.看到font=15px这里，这就是浏览Wap页面时字体的大小，我给它改成了15px。

2.另外看到选择器wml下面     Padding: 0;

这个就是最外边的白色边框所占的面积，改成0的话，这个讨厌的边框就没有了，

3.最后看到选择器card这里，有个属性padding:1em; 改成0.5em的话，文本离灰色的框的距离就会减小到一个字大小的一半了，这样又留出了更多的空间显示文本了。

这样文本显示的面积就很可观了：

如果你嫌背景有点灰的话，那你把它改成#FFFFFF吧。

其实这些都是CSS的问题，看来我没事情干的时候瞎琢磨了一下CSS终究还是有用的呀。此外还有很多属性值，懂CSS的马上就反应过来了，仔细调试并反复试验的话，很多令人震撼的效果就会产生了～

注意这个wml.css只能自定义浏览Wap页面的效果，对普通的网页是无效的。

  • 怎样改变Opera Mobile 10里面Fast Dial（快速拨号）里面的网址？ Opera启动后会有一个九宫格的常用网址快捷入口，但是添加了一个网址，以后要修改怎么办呢……嘿嘿，类似于触摸手机的操作，选中你要修改的那个格子，不要点击，而是按住它2秒钟，就会出来一个神奇的菜单了，或者按1也行！这样Fsat Dial也可以随意折腾了。
  • 如何在Opera Mobile 10里面在新窗口（新标签页）中打开链接？ 大家知道桌面级的浏览器，在新窗口或者新标签页里面打开链接是最基本的功能了，既然是“桌面”级的浏览器，Opera Mobile 10自然也是支持的。方法是，鼠标移到链接上面，同样不要点击，而是按住不动，接着就会出现一个神秘菜单了……在Opera里面，很多地方长按都出出现意想不到的好东西，不得不佩服Opera Mobile的创意啊，至少在S60系统上面是走到前列了。

总结：Opera还是非常不错的浏览器……如果选择特效和页面效果，那非Opera Mobile莫属；如果追求更多更实用的功能和更快的速度的话，那就继续UCWeb吧。
